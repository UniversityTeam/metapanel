# metapanel
Pretty and customizable admin panel module for impress

# Live demo
Live demo avalible on [heroku](https://metapanel.herokuapp.com/)

# Try it yourself
```
git clone https://github.com/UniversityTeam/metapanel
cd metapanel
npm i
sudo node server.js
```
Then check [localhost](http://localhost)
